# Specification

## Summary
**Goal:** Build a static single-page restaurant landing page for Eatery Burns Lake with a warm, rustic visual theme.

**Planned changes:**
- Apply a warm amber/terracotta/deep wood-brown color palette with serif/slab-serif headings and sans-serif body text throughout the entire page
- Build a Hero section with headline "🔥 Cozy Dining Experience in the Heart of Burns Lake", subheading, a "View Menu" scroll button, and a "Call Now" tel: button; use the hero background image as full-width background
- Display the restaurant logo in the top navbar/header
- Build an About Us section with "Welcome to Eatery Burns Lake" heading, description copy, and three feature callouts (rating, pricing, live music)
- Build a Menu section (with anchor ID) showing four category cards, a price range note, and a call-for-specials prompt
- Build a "Why Visit Us?" Experience section with four highlighted features and a closing tagline
- Build a Location section with address, clickable phone number (tel:+12502515445), hours, and highway access note
- Build a Call to Action section with prominent clickable phone number, heading, subtext, and closing line

**User-visible outcome:** Visitors see a fully styled, single-page restaurant landing site for Eatery Burns Lake where they can read about the restaurant, browse menu categories, find location/hours, and call directly from any section.
